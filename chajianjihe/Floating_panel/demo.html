Basic example using jsPanel

Example 1：创建一个默认的jsPanel:

$.jsPanel();

Example 2：创建一个带标题，主题样式和初始定位的jsPanel:

$.jsPanel({
    position: {
        left: 200,
        top:  100
    },
    title:    'jsPanel theme info',
    theme:    'info'
});
